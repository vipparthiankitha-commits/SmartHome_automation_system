# Smart Home Automation System Simulation
## Overview:
This project simulates a Smart Home Automation System using ESP32 in Wokwi. It monitors temperature and humidity, detects motion, and controls appliances through a virtual dashboard.
## Features:
•	Temperature Monitoring
•	Humidity Monitoring
•	Motion Detection
•	LED Control
•	IoT Dashboard Integration
## Tools/Technologies Used:
•	Wokwi simulator
•	ESP32
•	Arduino IDE
•	C/C++ (Arduino)
•	Blynk IoT Platform
## Project Structure:
•	Arduino code
•	Circuit Diagram
•	Simulation screenshots
•	Dashboard screenshots
•	Project report
## How to Run:
1.Open project in Wokwi.
2.Connect the project to Blynk.
3.Run the simulation.
4.Monitor Sensor values and control appliances
